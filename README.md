## Instalar 

Para utilizar o teste instale o gerenciador de ambientes [Pipenv](https://pipenv.pypa.io/en/latest/) veja o passo a passo abaixo:

1. Instalar o pipenv

```bash

pip install pipenv

```

2. Instalar as dependencias

```bash

pipenv install

```

3. Iniciar o ambiente virtual

```bash

pipenv shell

```
